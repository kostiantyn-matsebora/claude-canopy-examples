# Change Categories

The following categories are the only valid section headings inside a version entry.
Do not invent new categories.

| Category | When to use |
|----------|-------------|
| `Added`   | New features or capabilities |
| `Changed` | Changes to existing functionality (refactors, perf, chore) |
| `Fixed`   | Bug fixes |
| `Removed` | Features or files deleted from the project |
| `Security` | Vulnerability fixes or security-related changes |
