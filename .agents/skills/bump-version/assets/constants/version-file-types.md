# Version-Bearing File Types

Patterns searched at the project root. Matched files are reported as `files_to_update` in the explore output.

| Pattern | Type identifier |
|---------|----------------|
| `package.json` | `package.json` |
| `pyproject.toml` | `pyproject.toml` |
| `Cargo.toml` | `Cargo.toml` |
| `go.mod` | `go.mod` |
| `version.txt` | `version.txt` |
| `*.csproj` | `csproj` |
