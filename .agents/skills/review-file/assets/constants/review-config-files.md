# Review Config Files

Config files detected at the project root to establish the project's coding standards.

| Filename pattern | Purpose |
|---|---|
| `.eslintrc*`, `eslint.config.*` | JavaScript/TypeScript linting |
| `tsconfig.json` | TypeScript type-check config |
| `pyproject.toml` | Python tooling config |
| `.editorconfig` | Editor formatting rules |
